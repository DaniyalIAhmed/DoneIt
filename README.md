# DoneIt

This is an `Expo` project created with `create-expo-app`.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo
   ```


## Environment Variables

```bash
CONVEX_DEPLOYMENT=dev:your-convex-cloud
EXPO_PUBLIC_CONVEX_URL=https://your-convex-cloud
```
